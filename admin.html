🎨 Painting Manager

Private admin tool — manage your paintings without touching code.

How it works (3 steps):
1. Add, edit, delete or mark paintings as sold below.
2. Click Download paintings.json when done.
3. On GitHub, open your repo → data folder → paintings.json → click the ✏️ pencil (or "Replace file") → paste/upload the new file → Commit changes. The website updates in about a minute.

📷 New painting photos: upload the image file to the images/paintings folder on GitHub (Add file → Upload files), and use the same filename here.
